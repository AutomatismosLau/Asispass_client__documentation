ANEXO A: DESCRIPCIÓN TÉCNICA DEL SOFTWARE ASISPASS
═══════════════════════════════════════════════════════════════════════════════

A.1. Descripción General
Asispass es una plataforma de software como servicio (SaaS) desarrollada por AutomatismosLau LTDA., diseñada para el registro, seguimiento y gestión de las marcaciones de asistencia de trabajadores en empresas.

A.2. Funcionalidades Principales
• Registro de marcaciones de entrada y salida mediante conexiones remotas o dispositivos biométricos.
• Gestión de diferentes tipos de marcaciones (entrada, salida, inicio de colación, término de colación, etc.).
• Generación automática de reportes y estadísticas de asistencia.
• Panel de administración para gestión de usuarios, horarios y permisos.
• Integración con dispositivos de lectura biométrica (huellas digitales, reconocimiento facial).
• Geolocalización para marcaciones remotas (según configuración).
• Exportación de reportes en formatos estándar (CSV, Excel, PDF).
• Validación conforme a los requisitos de la Dirección del Trabajo de Chile.

A.3. Arquitectura Técnica
• Modelo: Multi-tenant
• Infraestructura: Amazon Web Services (AWS)
• Seguridad: Encriptación TLS/SSL en tránsito, encriptación en reposo
• Disponibilidad: Alta disponibilidad con redundancia geográfica
• Acceso: Web browser, aplicaciones móviles (si aplica)

A.4. Requisitos Técnicos del Cliente
• Conexión a Internet estable
• Navegador web actualizado (Chrome, Firefox, Safari, Edge)
• Dispositivos biométricos compatibles (si se utiliza modalidad presencial)
